CSS Cheat Sheet!!!!


Border

Define un estilo ante todos los bordes del objeto seleccionado

Border Left

Define especificamente el borde de la izquierda del objeto seleccionado

Border Right

Lo mismo pero con el de la derecha.

Border Top

Lo mismo pero con el de arriba.

Border Bottom

Lo mismo pero con el de abajo.

Border Radius

Define el radio de las esquinas del objeto seleccionado.

Background Image

Define una imagen de fondo para el elemento seleccionado.

Color

Define un color de fondo para el elemento seleccionado.

Font

Permite especificar valores relacionados con la fuente de un texto.

Font Family

Especifica la fuente de un texto.

Font Style

Define si una fuente debe aparecer normal, italic, bold, etc. acorde al estilo de la familia seleccionada

Font Weight

Define el grueso de la fuente de un texto.

Width and Height

Definen la altura y anchura de un elemento

List Style

  • hola
  • define el estilo de los bullets de una lista
  • a esta lista se le puso el estilo square

Margin

Ajuste automático de los bordes acorde a los elementos presentes

Padding

Define el espacio de relleno que tendran los bordes ante los elementos.

Overflow

Define si se recorta contenido o como se muestra en caso de que este supere el espacio permitido del objeto

Display

Especifica como un elemento será mostrado o tratado.

Position

Define como un elemento es posicionado dentro de un documento.


Filter

Define elementos visuales a un elemento.
